# SatSolverDpll
Реализация алгоритма Dpll с загрузкой КНФ-формулы из файла в формате DIMACS

### Входные файлы

В директории 'cnf' лежат примеры файлов с КНФ-формулами.

sat:

* lab2ex.cnf
* simple_v3_c2.cnf
* quinn.cnf
* lec2ex.cnf
* par8-1-c.cnf
* aim-50-1_6-yes1-4.cnf

unsat:

* hole6.cnf
* dubois20.cnf
